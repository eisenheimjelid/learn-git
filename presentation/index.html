    5. Caracteristicas

       

       

      Mecanismo de almacenamiento de los elementos que deba gestionar (ej. archivos de texto, imágenes, documentación...).

      Posibilidad de realizar cambios sobre los elementos almacenados (ej. modificaciones parciales, añadir, borrar, renombrar o mover elementos).

      Registro histórico de las acciones realizadas con cada elemento o conjunto de elementos (normalmente pudiendo volver o extraer un estado anterior del producto).

    4. clone // checkout

       

       

      clone - Clonar y/o descargar un código desde github.

      checkout - Dentro de un mismo repositorio cambiar a un tag o branch.

    5. commit

       

       

      commit - Anotar o crear una revisión al repositorio local.

    6. pull // push

       

       

      pull - Descargar de un repositorio remoto las anotaciones o cambios.

      push - Subir a un repositorio remoto las anotaciones o cambios.

    7. branch // tag

       

       

      branch - Crear una rama en el repositorio local.

      tag - Crear una etiqueta en el repositorio local.

    8. Hooks

       

       

      hooks - Disparador para automatizar procesos y/o actividades rutinarias.
